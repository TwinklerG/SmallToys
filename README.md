# nLLM(文心雕龙)

## 内容概述

四名南大大一学生开发的文化主题安卓应用

包含诗文回顾，AI问答，历史回顾，飞花令等内容

希望使用者可以在闲暇时光徜徉在文化的海洋

## 使用说明

团队提供两个版本供选择

### 开发版

开发版是正是本开源仓库，包含所有源代码和资源文件，在release里对应for_developers版本

也可供开发者测试自己的文心千帆API

### 用户友好版

无需API配置，开箱即用的特别定制版，在release里对应use-friendly版本
